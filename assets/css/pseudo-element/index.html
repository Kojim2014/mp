Pseudo Element

Mengenal Pseudo-Element

CSS pseudo-element digunakan untuk menghias bagian khusus dari suatu elemen.

✎ Untuk pseudo-element kita menggunakan 2 titik-dua ::
1. After dan Before

Kedua pseudo element ini digunakan untuk memasukkan konten, baik sebelum ataupun setelah elemen tersebut.

dengan ::selection pseudo-element, user atau pengguna website akan melihat efek ketika memilih atau memblok text. Opsi property untuk selection: color, background, cursor, dan outline.
