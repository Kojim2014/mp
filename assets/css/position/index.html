Mengatur Posisi

Ada beberapa variasi posisi pada CSS, masing-masing mempunyai sifat khusus yang perlu kita perhatikan.

✎ Menambahkan property atas, bawah, kiri dan kanan tidak berlaku pada posisi static.
1. Static

Secara automatis, elemen HTML kita memiliki nilai static untuk property posisi, kita tidak bisa memasukkan nilai atas,bawah, kiri dan kanan untuk ini.

4. Absolut

Posisi elemen akan tergantung dari parent-element atau di dalam mana elemen ini berada, hanya jika parent elemennya tidak bernilai static. Jika bernilai static, maka ia akan tergantung pada browser sama seperti fixed.

5. Z-Index

Bayangkan jika website kita tiga dimensi, atau bukan hanya berada pada koordinat x dan y, tapi juga ada koordinat z, dimana kita bisa mengatur elemen mana yang berada di depan elemen lain. Setiap elemen bisa memiliki nilai negatif atau positif.
