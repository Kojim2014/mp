Mengenal Float dan Clear

Float adalah bagian penting dari pengaturan posisi pada CSS. Dengan Float kita mendorong elemen ke kanan ataupun ke kiri dan memberi ruang untuk elemen lain mengisi sisa baris tersebut selebar browser atau viewport.

✎ Elemen yang diberi property float akan mempengaruhi elemen setelahnya.
