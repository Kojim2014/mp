    Front End
    Pintu pertama yang harus kita buka untuk mulai belajar cara membuat website website, yaitu front-end (bagian depan).
    Membuat Portofolio
    Seri kelas mulai dari mendesain website, mengubah menjadi HTML, CSS dan Javascript. Tidak ketinggalan cara untuk mengubah website menjadi responsive.

    Javascript
    Kuasai javascript untuk menjadi ahli di bagian pengembangan 'front end website'.
    Java
    Salah satu bahasa program yang sangat melegenda, pondasi untuk membuat aplikasi desktop, mobile ataupun back-end pada website.
