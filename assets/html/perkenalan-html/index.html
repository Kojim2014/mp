HTML singkatan dari HyperText MarkUp Language adalah bahasa yang digunakan untuk menampilkan halaman website pada browser (Firefox, Chrome, Internet Explorer dll.). Sama seperti bahasa Indonesia yang kita pakai untuk berkomunikasi dengan orang Indonesia, HTML adalah bahasa yang kita gunakan untuk berkomunikasi dengan browser.
✎ File HTML terdiri dari berbagai tag HTML, kita akan belajar masing-masing tag HTML di Sekolah Koding
Tag adalah kosa-kata dari HTML yang dipakai untuk membuat Website. Tag biasanya terdiri dari tag pembuka dan tag penutup. Keduanya menggunakan tanda kurung lancip buka '<' dan tutup '>'. Untuk tag penutup ia ditambahkan dengan garis miring '/' . Sementara elemen HTML adalah istilah yang terdiri dari gabungan tag pembuka, konten dan tag penutup.
