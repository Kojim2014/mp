Angka

Tipe Data Angka

Tidak seperti bahasa program lainnya, javascript hanya memilih satu tipe data angka.

✎ Tipe data angka (numbers) dalam javascript memiliki 64 bit.
1. Penulisan

Untuk menggunakan tipe data angka, cukup menuliskan angkanya dan ditambah tanda titik untuk angka desimal (di belakang koma). Untuk hexadecimal kita cukup menulis 0x telebih dahulu.

2. Mengubah basis

Pada default atau secara automatis tipe data angka memiliki basis 10 desimal. Tapi kita bisa mengubahnya dengan metode .toString() dan memasukkan basis di antara tanda kurung.
